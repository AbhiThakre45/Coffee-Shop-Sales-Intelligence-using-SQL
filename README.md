# ☕ Coffee Shop Sales Intelligence using SQL

---

## 📌 Project Overview

BrewMetrics is a SQL-based project that analyzes coffee shop sales data to generate business insights on revenue trends, product performance, and store efficiency.

---

## 🎯 Problem Statement

To analyze coffee shop sales data and identify key patterns in revenue, products, and store performance for better decision-making.

---

## 🛠️ Tools Used

* SQL (MySQL)
* Data Cleaning
* Data Modeling
* Exploratory Data Analysis (EDA)

---

## 🧱 Data Modeling

The dataset was structured into three tables:

* **Products**
* **Stores**
* **Sales**

---

## 🔄 Data Preparation

* Handled null values and duplicates
* Converted date formats
* Created new columns:

  * Revenue
  * Month
  * Hour

---

## 📊 Analysis Performed

* Revenue & Order Analysis
* Top Products & Category Performance
* Store-wise Sales Analysis
* Monthly & Hourly Trends
* High Revenue Transactions

---

## 🔍 Key Insights

* Top products generate most of the revenue
* Certain categories dominate sales
* Store performance varies by location
* Sales peak at specific hours
* Revenue shows monthly variation

---

## 📈 Conclusion

SQL analysis helped uncover key sales patterns and business insights that can support better product and store-level decisions.
